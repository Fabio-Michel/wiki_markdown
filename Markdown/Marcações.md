## Texto Mono-Espaçado Inline (Apresentação de código)

**Sem coloração**

##### Python
```
def my_function():
    print("Hello from a function")
```
##### JavaScript
```
function my_function(numero) {
  return numero * numero;
}
```

##### Python
```python
def my_function():
    print("Hello from a function")
```
##### JavaScript
```js

  return numero * numero;
}
```
